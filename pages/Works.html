4Côtés.com @4cotes

Projet de la startup 4Côtés.com qui a pour but d'offrir un service de coupe de materiaux d'ébinisterie professionnel à moindre coût et totalement sur mesure.

La problématique à été dans l'interface de coupe sur mesure, de nombreux test on été effectués afin de rendre cette partie intuitive et facile d'accès à une clientel peux habituée à ce genre d'achat.

Techno du projet :

  • WordPress
  • Javascript
  • JQuery
  • PHP

DppLog.fr @DppLog

Refonte complète graphique d'un site web client en full responsive. Création identité graphique ainsi que nouveau logo d'entretprise.

Discussion avec le client pour la direction graphique à prendre et simplicité choisi pour être en accord avec la clientèle ciblée. Nombreux tests reponsive multi-supports et mandat de conseil en communication visuel.

Techno du projet :

  • Html5
  • Javascript
  • JQuery
  • Css3

Nouméa

Projet d'apprentissage WP pour ma reconversion front-end.

Création d'un site fictif de l'office du tourisme de Nouméa. Programmation culturel, activité sportive, plan de la ville... le tout récupérer via l'api officielle.
Fun fact: Le site à eu tellement de succès que la vraie office du tourisme fait une demande pour supprimer le site en ligne afin de récupérer son flux de visite habituelle

Techno du projet :

  • WordPress
  • Javascript
  • JQuery
  • PHP

Vloc

Projet d'apprentissage Jquery pour ma reconversion front-end.

Objectif de l'exercice: apprendre à gérer la récupération de données via API. Le site est une carte de location de vélo ce situant sur la ville de Nantes (France). La carte est totalement interactive et permet de voir en direct le nombre de vélos disponibles à la location ainsi qu'un service de réservation en direct (délai de 30min pour aller validation la location à la borne)

Techno du projet :

  • HTML5
  • Javascript
  • JQuery
  • PHP
